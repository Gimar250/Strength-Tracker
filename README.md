## рџ› пёЏ Build It Yourself

Follow these steps to get a local developer copy up and running.

### Prerequisites

* **Android Studio:** [Latest version](https://developer.android.com/studio) recommended.
* **Android SDK:** Configured within Android Studio.
* **Hardware:** A physical Android device or a configured Android Emulator.

### Setup and Execution

1. Open your terminal or command prompt and clone the repository:
```bash
git clone https://github.com/Gimar250/Strength-Tracker.git
```

2. Open **Android Studio**.
3. Select **File > Open** and navigate to the cloned `Strength-Tracker` directory.
4. Allow Gradle to sync and build the project dependencies.
5. Connect your Android device via USB (ensure USB Debugging is enabled) or start an Android Emulator.
6. Click the green **Run** button (`Shift + F10`) in the toolbar.
7. Select your device/emulator and wait for the APK to build and install.
